#APICloud Javascript Framework

##APICloud 脚本框架 api.js

我们为 APICloud 开发者提供了一个可选的前端脚本框架，该框架可与任何第三方前端框架混用。

api.js 借鉴了 Zepto.js，去掉了开发 APICloud 应用时不必要的方法，只提供最基础的 Javascript 函数，做到尽可能的轻量和简洁。
